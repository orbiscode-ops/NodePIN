# Repocket

مشاركة bandwidth مقابل USD — يُدفع عبر PayPal/Crypto.

## المتطلبات
- حساب على [repocket.co](https://link.repocket.co/nodepin)
- انسخ الـ API Key من الإعدادات

## المتغيرات
| المتغير | الوصف |
|---------|-------|
| `REPOCKET_EMAIL` | البريد الإلكتروني |
| `REPOCKET_API_KEY` | API Key من لوحة التحكم |

## التفعيل
```
ENABLED_NETWORKS=repocket
```

## الأرباح
❌ تظهر على [app.repocket.co/dashboard](https://app.repocket.co/dashboard) فقط
