# Gradient Network

شبكة AI bandwidth لامركزية — تدفع GRAD tokens. مرحلة مبكرة والدخول الآن يعني نقاط أكثر.

## التسجيل

1. اذهب إلى <https://app.gradient.network> وأنشئ حساباً
2. ضع بيانات الحساب في `.env`:

```env
GRADIENT_EMAIL=your@email.com
GRADIENT_PASS=your_password
```

## تفعيل الشبكة

```env
ENABLED_NETWORKS=...,gradient
```

## ملاحظات

- مشروع ضخم في مرحلته الأولى — الدخول الآن يضمن نقاط سيزون أول
- لا يوجد API محلي — راجع نقاطك على <https://app.gradient.network/dashboard>
- العملة: **GRAD Token** — تُوزَّع عند الإطلاق الرسمي
- Docker image: `community/gradient-node:latest`
