# Traffmonetizer

مشاركة bandwidth مقابل USD — يُدفع عبر BTC/USDT/PayPal.

## المتطلبات
- حساب على [traffmonetizer.com](https://traffmonetizer.com/?aff=nodepin)
- انسخ الـ Token من لوحة التحكم

## المتغيرات
| المتغير | الوصف |
|---------|-------|
| `TRAFFMONETIZER_TOKEN` | Token من لوحة التحكم |

## التفعيل
```
ENABLED_NETWORKS=traffmonetizer
```

## الأرباح
❌ تظهر على [traffmonetizer.com/dashboard](https://traffmonetizer.com/dashboard) فقط
