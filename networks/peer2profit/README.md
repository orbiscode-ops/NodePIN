# Peer2Profit

مشاركة bandwidth مقابل USD.

## المتطلبات
- حساب على [peer2profit.com](https://peer2profit.com)

## المتغيرات
| المتغير | الوصف |
|---------|-------|
| `PEER2PROFIT_EMAIL` | البريد الإلكتروني |

## التفعيل
```
ENABLED_NETWORKS=peer2profit
```

## الأرباح
❌ تظهر على [peer2profit.com/cabinet](https://peer2profit.com/cabinet) فقط
