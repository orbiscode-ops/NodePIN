# IPRoyal Pawns

مشاركة bandwidth مقابل USD — يُدفع عبر PayPal/Crypto.

## المتطلبات
- حساب على [pawns.app](https://pawns.app?r=nodepin)

## المتغيرات
| المتغير | الوصف |
|---------|-------|
| `IPROYAL_EMAIL` | البريد الإلكتروني |
| `IPROYAL_PASS` | كلمة المرور |
| `IPROYAL_DEVICE` | اسم الجهاز (افتراضي: nodepin) |

## التفعيل
```
ENABLED_NETWORKS=iproyal
```

## الأرباح
❌ تظهر على [pawns.app/dashboard](https://pawns.app/dashboard) فقط
