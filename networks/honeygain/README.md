# Honeygain

مشاركة bandwidth غير مستخدم مقابل USD.

## المتطلبات
- حساب على [honeygain.com](https://r.honeygain.me/nodepin) (مجاني)

## المتغيرات
| المتغير | الوصف |
|---------|-------|
| `HONEYGAIN_EMAIL` | البريد الإلكتروني |
| `HONEYGAIN_PASS` | كلمة المرور |
| `HONEYGAIN_DEVICE` | اسم الجهاز (افتراضي: nodepin) |

## التفعيل
```
ENABLED_NETWORKS=honeygain
```

## الأرباح
✅ تظهر مباشرة في NodePIN Dashboard (عبر Honeygain API)
