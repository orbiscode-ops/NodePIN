الخوادم المضافة (VPS Servers)

اسم السيرفر عناوين الـ IP المرتبطة حالة التحديد إجراءات
لا توجد خوادم مضافة حالياً. اضغط على زر الإضافة لإعداد خادمك الأول.

إعدادات النودات العامة

المتغيرات المتاحة: {hostname} اسم السيرفر، {type} نوع النود (wireguard, v2ray, openvpn).

إضافة خادم جديد

أدخل جميع الـ IP المخصصة لهذا السيرفر مفصولة بفاصلة ( , ).

إطلاق نود Sentinel جديد

✔️ تم إطلاق النود وتوليد المحفظة بنجاح!

تم تشغيل النود وربطه بالـ IP المحدد. لكي يعمل النود بشكل كامل، يجب عليك تمويل عنوان المحفظة بالـ DVPN.

احفظ هذه الكلمات في مكان آمن للغاية. لن يتم عرضها مرة أخرى أبداً!
